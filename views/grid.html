Layout Grid

The grid system is based off the flexboxgrid with a 12-column row base. Using it, you can specify different column widths, offsets, alignment, and distribution at xs, sm, md & lg size widths.



Basic Syntax

The correct format will always be row, column, content. 

            <div class="row">
                <div class="col-xs-12">
                    Column Content
                </div>
            </div>

Basic Responsive Structure

With responsive grids, you specify the width you want on specific display sizes (out of a grid of 12). Start with xs, then sm, md, lg. At minimum you need to specify col-xs-##, and only need to specify larger view widths if the columns are expected to change/adapt.

The correct format for grid classes are col-[[viewsize]]-[[# 1-12]]. For example col-sm-12 or col-lg-4

If your column widths do not add up to exactly 12, there will be whitespace left over.

 
 
 
 
 
 
 
 
 

Offset

Offset columns within a row and create whitespace.

 
 
 
 
 
 
            <div class="row">
                <div class="col-xs-offset-3 col-xs-9">
                    Offset Column
                </div>
            </div>

Auto Width Columns

While you can specify how many columns content takes up, you can also have a column fill any space is left in the row simply by not specifying a column width. Example col-xs

 
 
 
 
            <div class="row">
                <div class="col-xs-3">
                    3-width-column
                </div>
                <div class="col-xs">
                    Auto-width column fills remainder
                </div>
            </div>

Align Elements

Easy classes to align elements at the start, end, or center of a row, or the top, middle, or bottom of a column. You can use these to have different alignment on different view sizes.

.start-

 
 
            <div class="row start-xs">
                <div class="col-xs-6">
                    Row with elements aligned start
                </div>
            </div>

.center-

 
 
            <div class="row center-xs">
                <div class="col-xs-6">
                    Row with elements aligned center
                </div>
            </div>

.end-

 
 
            <div class="row end-xs">
                <div class="col-xs-6">
                    Row with elements aligned end
                </div>
            </div>

.top-

 
 
 
 
            <div class="row top-xs">
                <div class="col-xs-6">
                    Column aligned top
                </div>
            </div>

.middle-

 
 
 
 
            <div class="row middle-xs">
                <div class="col-xs-6">
                    Column aligned middle
                </div>
            </div>

.bottom-

 
 
 
 
            <div class="row bottom-xs">
                <div class="col-xs-6">
                    Column aligned bottom
                </div>
            </div>

Element Distribution

Add classes to distribute and space columns within a row.

.around-

 
 
 
            <div class="row around-xs">
                <div class="col-xs-6">
                    Row with space around elements
                </div>
            </div>

.between-

 
 
 
            <div class="row between-xs">
                <div class="col-xs-6">
                    Row with space between elements
                </div>
            </div>
