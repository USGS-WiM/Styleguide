WiM Style Guide

The WiM Style Guide should be used when developing or designing WiM applications. It includes classes to style Typography, Buttons, Form Elements, various UI Elements, and also has a built in grid system built on Flexbox.

Page Structure

The structure for each application will be different depending on what is used. There are specific styles for the generator, but if you're creating your own application from scratch, you can use the grid from here.

If you're working on a static site (non-application), this style guide includes some classes that will help with the layout. The same structure is used throughout this styleguide, so if you need a page to look like this, follow the code template below. 


                <div class="container">
                    <section>
                        <h2>
                            Section Header
                        </h2>
                        <p>
                            Section description or paragraph
                        </p>
                    </section>
                    <hr/>
                    <section>
                        ...
                </div>

Here are what the (optional) layout helpers do.

  • Class Description
  • .container Centered, max with 1000px, padding left & right
  • <section> Full width, margin above and below for spacing
  • <hr/> Dashed line break
