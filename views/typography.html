Links

If needed, you can have links that stand out more if you add the class .astrong to the link element.

Miscellaneous

Block quotes

If copying content from another source (blog, external website, etc) you should quote the text in a <blockquote> tag, and if necessary, give attribution using an <em> tag within the blockquote.

We provide science about the natural hazards that threaten lives and livelihoods; the water, energy, minerals, and other natural resources we rely on; the health of our ecosystems and environment; and the impacts of climate and land-use change. - USGS.gov homepage 
                    <blockquote>
                        We provide science about the natural hazards that threaten lives and livelihoods; the water, energy, minerals, and other natural resources we rely on; the health of our ecosystems and environment; and the impacts of climate and land-use change.
                        <em>- USGS.gov homepage</em>
                    </blockquote>

Helper Classes

Use these helper classes to modify text within an element without using inline styles.

  • Class Description
  • .alignc Center align text
  • .alignl Left align text
  • .alignr Right align text
  • .fullwidth Paragraphs default to max width of 680, use this class to override it.
  • .capitalize Capitalize every word
  • .uppercase Transform all text to uppercase
  • .lowercase Transform all text to lowercase
  • .smalltext Make all text smaller (13px) than the default (15px)
