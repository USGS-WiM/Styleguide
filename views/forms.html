Form Elements

Forms should always be wrapped in a form tag, and the form layouts shoud be created using the grid system. Each input should have a label element along side it.



Buttons

Standard Buttons

Simply using the <button> tag will style a button and make grey. To color buttons use the classes .btn-black, .btn-blue, .btn-red. 


                    <button>Standard</button>
                    <button class="btn-black">Action Button</button>
                    <button class="btn-blue">Info Button</button>
                    <button class="btn-red">Problem Button</button>
                    <button disabled>Disabled Button</button>

Large buttons

Large buttons are available, but should be used as little as possible. To make a button large, add the class .btn-lg to a button element. 

                    <button class="btn-lg">Large Button</button>

Link buttons

Link buttons are good to use when a button element is necessary, but it is less important. Look at the confirmation modal on the UI page for an example of how this should be used.
Link buttons can only be black or white. 


                    <button class="btn-link">Black Link Button</button>
                    <button class="btn-link white">White Link Button</button>

Other elements styled as buttons

You can turn basically any element (<div>, <a>, <input>, etc) into a button by adding the .button class to them.

Anchor Button
Div Button
                    <a class="button">Anchor Button</a>
                    <div class="button">Div Button</div>
                    <input type="button" value="Input button" class="button" />

Loading buttons

When an action is being performed, such as data loading from an API after a button click, use a loading button to show the user that something is happening.
To start a loading button, add the class .loading to the button after it is clicked. You should also set the button to disabled to prevent multiple clicks/submissions. 


                    <button class="loading">Click Me</button>

Text Inputs

Input elements will default to 100% width. To make them smaller, wrap them in a grid and set the width using columns. Forms and form elements should always be grouped alone in a div or column, only with a label, not other form elements.

Basic text inputs


Tip: To simplify code when using the grid, you can add the class .row to a <form> instead of having a separate div for the .row. 


                    <form class="row>
                        <div class="col-xs-12 col-md-6>
                            <label>Text Input</label>
                            <input type="text" placeholder="Search..."/>
                        </div>
                    </form>

Input Hints

Use the elements and classes below to help users understand what they need to be doing with a form.

Text input with helper

Only your house number and street.

                    <form class="row>
                        <div class="col-xs-12>
                            <label>Address</label>
                            <input type="text" placeholder="Type..."/>
                            <div class="input-helper">Only your house number and street.</div>
                        </div>
                    </form>

Invalid Item

That email is not valid.

                    <form class="row>
                        <div class="col-xs-12 form-invalid>
                            <label>Email Address</label>
                            <input type="text" value="hello usgs.gov"/>
                            <div class="input-helper">That email is not valid.</div>
                        </div>
                    </form>

Required Item

Add a label with the class .required above any input to mark it as required. The input should also have a required tag for form validation. 


                    <form class="row>
                        <div class="col-xs-12>
                            <label class="required">Full Name</label>
                            <input type="text" placeholder="John Smith" required/>
                        </div>
                    </form>

Checkboxes & Radio

Checkboxes and radio elements are tricky to style and position. They absolutely must be wrapped in a .row and .col- element to esure the label stays with the radio or checkbox.

Single line checkboxes and radios


Checkbox code
                    <form class="row>
                        <div class="col-xs-12 col-md-6>
                            <label for="inputname" class="chx">
                                <input type="checkbox" id="inputname"/>
                                <span>Checkbox Input</span>
                            </label>
                        </div>
                    </form>
Radio code
                    <form class="row>
                        <div class="col-xs-12 col-md-6>
                            <label for="inputname" class="rdo">
                                <input type="radio" id="inputname"/>
                                <span>Radio Input</span>
                            </label>
                        </div>
                    </form>

Multi-line checkbox or radio

Nest a <p> tag inside the <span> tag within the label of a checkbox or radio input. 


                    <form class="row>
                        <div class="col-xs-12 col-md-6>
                            <label for="inputname" class="rdo">
                                <input type="radio" name="inputname"/>
                                <span>
                                    <p>
                                       <b>By checking this box...</b>
                                       I agree to the WiM Privacy Policy and Terms of Use...
                                    </p>
                                <span>
                            </label>
                        </div>
                    </form>

Switches

Switches are simply checkboxes styled to look like switches. To use one, add the class .switch alongside a input type="checkbox". 


                    <form class="row>
                        <div class="col-xs-12 col-md-6>
                            <label for="switchname" class="switch">
                                <input type="checkbox" name="switchname"/>
                                <span>Switch</span>
                            </label>
                        </div>
                    </form>
