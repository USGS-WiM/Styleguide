UI Elements

UI Elements and transformations that can be used throughout applications.



UI Coloring

You can change the background and border color of any element by adding classes to it. This applies to any element, including the specific UI elements listed on this page.

.success
.info
.warning
.error

You can use more intense/vibrant colors by adding the class .intense along with the coloring class. Use these infrequently.

.success .intense
.info .intense
.warning .intense
.error .intense

Toast Notifications

Toast notifications should be used to show important information to the user, whether good or bad.
You will need to write your own JS code to show/hide the notifications. It should disappear automatically after ~3-5 seconds and/or when clicked.
You could also use the toast element as a permanent banner.


Data has been updated, now showing the most recent data.


You can also change the colors of toasts using the classes .success, .info, .warning, or .error. You can also use the .intense class on toast notifications.


Something went wrong. Please try again later. 


                    <div class="toast error">
                        Something went wrong. Please try again later.
                    <div>

Modals

Alert or confirm modal

Use this to either alert users that something has happened, or to make them confirm that they want to complete an action.


Are you sure?

Do you really want to clear the map?

No, Cancel


Tip: When labeling the buttons in a confirmation modal, make sure they say exactly what they will do. For example, using "No, Cancel" and "Yes, Clear It" instead of just "No" and "Yes". 


                    <div class="modal-wrapper">
                        <div class="modal">
                            <div class="modal-header">
                                Are you sure?
                                <div class="close-modal">
                                    <class="ion-close-circled"></i>
                                </div>
                            </div>
                            <div class="modal-body">
                                <p class="p-meta alignc">Do you really want to clear the map?</p>
                            </div>
                            <div class="modal-footer">
                                <a href="#">No, Cancel</a>
                                <button>Yes, Clear It</button>
                            </div>
                        </div>
                    </div>

UI Helper Classes

Transformation Helpers

Use these classes to quickly transform a UI element looks or behaves.

  • Class Description
  • .fullwidth Display: block & width: 100%
  • .invisible Visibility hidden
  • .hide Display none
  • .inline Display inline
  • .inlineblock Display inline-block
  • .overflowhidden Element overflow is hidden
  • .overflowauto Element overflow is auto/default
  • .overflowvisible Element overflow is visible
  • .overflowtext Cuts text off with an ellipsis. See example below.

With the class overflowtext, when text overflows a view, it will end with an ellipsis. You can see that if I keep typing and this text gets too long.


Spacing Helpers

Use these classes to quickly add spacing, margins, and padding to elements.

  • Class Description
  • <hr/> Dashed spacer line
  • .margin--xauto Applys margin: 0 auto, centering element
  • .margin--x0 Removes all margins
  • .padding--x0 Removes all padding
  • .margin--x[#] Equal margin on all sides
  • .padding--x[#] Equal padding on all sides
  • .margin-[side]--x[#] Margin applied only on [side]
  • .padding-[side]--x[#] Padding applied only on [side]

[side] is the side you want the margin or padding to be applied to. Your options are top, right, bottom, or left.

[#] is the level (0-6) of margin or padding you want. The levels are displayed below.


Level 1: 10px
Level 2: 18px
Level 3: 26px
Level 4: 34px
Level 5: 42px
Level 6: 50px

                    <!-- No margin or padding -->
                    <div class="margin--x0 padding--x0">
                     
                    <!-- Level 2 (18px) margin on all sides -->
                    <div class="margin--x2">
                     
                    <!-- Level 1 (10px) padding on all sides -->
                    <div class="padding--x1">
                     
                    <!-- Level 3 (26px) margin on top only -->
                    <div class="margin-top--x3">
                     
                    <!-- Level 5 (42px) padding on left only -->
                    <div class="padding-left--x5">
                     
                     
                    <!-- Level 5 (42px) padding on right, level 3 padding on top, & level 6 margin on bottom -->
                    <div class="padding-right--x5 padding-top--x3 margin-bottom--x6">
